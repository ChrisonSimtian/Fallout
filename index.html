Fallout

Build automation for C#/.NET — the hard-fork successor to NUKE.

Site under construction. The original nuke.build site is offline. Fallout continues NUKE's mission under new maintenance — full documentation is being rebuilt at this domain. For now, everything lives on GitHub.

Where to find things

  • Changelog — what's shipped
  • Releases — version history

Migrating from NUKE

If you're an existing NUKE user, a migration guide is in progress (tracked in #37). The short version: Nuke.* namespaces and packages are being renamed to Fallout.*, with shim packages and tooling planned to make the transition non-disruptive.

What's coming here

Full documentation, guides, and the rest of what nuke.build used to host is being ported in #41. Watch the repo for updates.
